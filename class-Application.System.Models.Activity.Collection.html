Class Collection

Activity stream collection model class

An Activity Stream is a collection one or more individual activities. This provides methods for modelling a collection of activities within a stream. A "collection" is a generic list of Objects of any object type. The objectType of each item in the collection MAY be omitted if the type of object can be established through context. The collection is used primarily as the root of an Activity Streams
