Platform Manual

ver.02|January|2011

Tuiyosocial is a a social networking platform enabling you to build exciting social communities; from small academic workgroups, to towns and cities of any size. With the changing nature of the Internet and the social networking landscape, we seek to provide a useful tool with which to give you total control over your own social niche and data. Tuiyosocial is built by a very small team of enthusiasts, and not backed by any corporations or agenda.

By using open source technologies, the platform is completely open and free, packed with of all the main features you'd expect of a social network, including extensive member management tools, profiles, relationship and interaction management and many more.

Keeping Everything up-to-date

All of CodeMirror is released under a MIT-style license. To get it, you can download the latest release or the current development snapshot as zip files. To create a custom minified script file, you can use the compression API.

We use git for version control. The main repository can be fetched in this way:

git clone http://marijnhaverbeke.nl/git/codemirror2

CodeMirror can also be found on GitHub at marijnh/CodeMirror2. If you plan to hack on the code and contribute patches, the best way to do it is to create a GitHub fork, and send pull requests.

Support and bug reports
