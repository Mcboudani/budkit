Class Uri

A library providing URI and URL parsing capability

The main purpose of this class is to automatically determine the key components pertaining to identifying the requested resource as well as build resource identifiers to system resources and actions. Whilst, tt does not provide any routing capability, this class is crucial to routing user queries to appropriate actions. cf \Library\Router
